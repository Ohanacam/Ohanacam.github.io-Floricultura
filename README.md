# Landing Page - Floricultura

Uma landing page responsiva para uma floricultura fictícia, desenvolvida utilizando **HTML5** e **CSS3**. O projeto apresenta os principais serviços, produtos e informações de contato da empresa em um layout moderno, organizado e responsivo.

![Preview da Landing Page](images/LandingPageGif.gif)

## Funcionalidades

- Navegação por âncoras
- Seção Hero
- História da floricultura
- Seção de serviços
- Catálogo de produtos
- Galeria de imagens
- Informações de contato
- Mapa integrado com Google Maps
- Layout responsivo
- Ícones do Font Awesome
- Tipografia personalizada com Google Fonts

## Tecnologias

- HTML5
- CSS3
- Google Fonts
- Font Awesome
- Google Maps Embed

## Como executar

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/flores-da-ana.git
```

Acesse a pasta do projeto:

```bash
cd flores-da-ana
```

Abra o arquivo `index.html` em qualquer navegador.

Ou utilize a extensão **Live Server** no Visual Studio Code para executar o projeto localmente.
